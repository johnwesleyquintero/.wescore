Installation

Install Wescore as a development dependency using your preferred package manager:

npm install --save-dev wescore
yarn add --dev wescore
pnpm add --save-dev wescore
bun add --dev wescore

Ensure your project also has your chosen tools installed (peer dependencies):

  • Code formatter (e.g., Prettier)
  • Linter (e.g., ESLint)
  • Type checker (e.g., TypeScript, if applicable)
  • Build tools (if applicable)
